---------------------------------


Далее, мигрируем базу:

```
php yii migrate --migrationPath=vendor/halumein/yii2-like-module/migrations
```

Подключение и настройка
---------------------------------
В конфигурационный файл приложения добавить модуль like

```php
    'modules' => [
        'like' => [
            'class' => 'halumein\like\Module',
        ],
        //...
    ]
```
